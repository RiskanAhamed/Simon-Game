# 🎮 Simon Game

A classic memory sequence game built with vanilla web technologies. Test and improve your memory by repeating the ever-increasing pattern of colors and sounds!

## 🔗 Live Demo
Play the game live here: **[Simon Game Live Demo](https://github.io)**

---

## 🕹️ How to Play
1. Press any key on your keyboard to start the game.
2. The game will flash a button and play a sound (e.g., Green).
3. Click the same button that flashed.
4. Each time you get the sequence right, the game adds one more random color to the pattern.
5. If you click the wrong button, the game ends. Press any key to restart!

---

## 🛠️ Tech Stack
* **HTML5** - Game structure and elements.
* **CSS3** - Layout, animations, and custom styling.
* **JavaScript (ES6)** - Game logic, DOM manipulation, and audio event handling.
